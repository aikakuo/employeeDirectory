## Installation
  The application will be invoked by using the following command:
  `npx create-react-app` 
  `my-app`
  `cd my-app`
  `npm start`
  bash and node index. 
  If node_modules missing axios  `npm i axios`.
  - `axios` fetches data from the RandomUser API
